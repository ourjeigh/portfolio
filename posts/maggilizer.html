MaggiLizer Wwise Plugin


This has been an on and off project for me to practice some DSP fundamentals, which I started after watching the Wwise Up On Air discussing how to get started with Wwise Plugin development.



MaggiLizer is an homage to one of my favorite VST plugins, SoundToys Crystallizer.



The plugin is a hobby and work in progress up on GitHub. I recently spent some time rewriting a good portion of the plugin to utilize more formal data structures, particularly a ring buffer for output since I wasn't familiar with the technique when I got started years ago.
